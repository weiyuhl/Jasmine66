# Production Skeleton

A clean, production-ready Android application skeleton based on modern architecture and best practices.

## Features
- **Modern UI**: Built with Jetpack Compose.
- **Architecture**: Follows the official Android architecture guidelines.
- **Dependency Injection**: Hilt for DI.
- **Data Layer**: Room for local storage, Retrofit for networking (currently placeholder).
- **Navigation**: Type-safe navigation with Navigation 3.

## Getting Started
- Open in Android Studio (Ladybug or later).
- Run the `debug` or `release` build.
